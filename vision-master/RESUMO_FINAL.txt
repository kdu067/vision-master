🚀 COMO USAR AGORA
════════════════════════════════════════════════════════════════════════════════

1️⃣ EXTRAÇÃO E PREPARAÇÃO
   $ unzip agrovision_ia.zip
   $ cd agrovision_ia
   $ python -m venv .venv
   $ .\.venv\Scripts\Activate.ps1  (Windows)
   $ pip install -r requirements.txt

2️⃣ INSTALAR OLLAMA (em outro terminal)
   $ ollama pull llama3
   $ ollama serve

3️⃣ RODAR O PROJETO (no primeiro terminal)
   $ python -m uvicorn app:app --reload
   $ Abrir http://127.0.0.1:8000

4️⃣ USAR NO PAINEL
   ├─ Ver câmera ao vivo
   ├─ Conferir eventos detectados
   ├─ Conversar com agente
   └─ Fazer perguntas sobre monitoramento

⚙️ CUSTOMIZAÇÕES SIMPLES
════════════════════════════════════════════════════════════════════════════════

Trocar Câmera:
  .env → CAMERA_SOURCE=https://nova-camera.com/stream

Trocar Modelo YOLO:
  services/video_monitor.py → YOLO("yolo11m.pt")

Trocar Modelo Ollama:
  .env → OLLAMA_MODEL=mistral

Customizar Agente:
  services/monitoring_agent.py → AGENT_PROFILE = ...

Adicionar Classe de Detecção:
  services/config.py → TARGET_CLASSES = [...]
